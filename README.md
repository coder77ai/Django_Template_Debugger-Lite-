# Django Template Debugger (AI-Assisted)

An enhanced debugging system for Django templates that detects missing variables and provides intelligent suggestions.

## 🚀 Features
- Detects missing template variables
- Suggests closest matches (NLP-based)
- Strict mode (raise errors)
- Flexible mode (warnings)

## 🛠 Tech Stack
- Python
- Django
- difflib (for suggestions)

## ⚙️ Setup

```bash
git clone <repo>
cd django-template-debugger
pip install -r requirements.txt
python manage.py runserver
